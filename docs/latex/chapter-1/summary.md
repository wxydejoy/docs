# 第一章 LaTeX的出现与发展

## 本章小结

本章回顾了TeX和LaTeX的出现与发展历程，同时讨论了随着信息技术的快速发展和互联网的深度普及，LaTeX在线系统正在作为新生事物改变着人们的创作习惯。对于LaTeX用户来说，互联网提供了很多便利，我们可以通过一些著名的LaTeX问答社区和开源社区获取大量的学习素材，这些资源能帮助用户更好地掌握和使用LaTeX。
